# MySQL常用命令

注意：服务的操作需要管理员权限运行 

- 关闭MySQL服务：net stop MySQL服务名(一般是mysql)

- 启动MySQL服务：net start MySQL服务名

- 登陆MySQL数据库：mysql -u用户名 -p密码

  ​                                    mysql -h要连接的ip地址 -u用户名 -p要连接的数据库密码

  ​                                    mysql --host=要连接的ip地址 -u用户名 -p要连接的数据库密码

- 退出MySQL数据库：exit   或 quit
